Bekannte Fallen & Regeln

🐍
PowerShell & UTF-8
Get-Content | Set-Content liest als Windows-1252 und schreibt kaputt. Immer Python verwenden: open(path, encoding='utf-8')
🔑
Supabase API Key
Neuer publishable Key (sb_publishable_...) liefert 401 bei Edge Functions. Nur den Legacy JWT Anon Key verwenden!
💥
Variable supabase
CDN registriert sich als window.supabase. Eigene Variable heisst immer sb.
🗄️
Kein anonymous-Feld
In exercise_responses gibt es keine anonymous-Spalte! Anonym = alias === null. Sonst HTTP 400.
⚡
PowerShell &&
Geht nicht. Befehle mit ; trennen.
📁
Filesystem write_file → EPERM
Auf Windows EPERM beim direkten Schreiben. Python-Skript nach E:\fix_xxx.py schreiben und via Shell ausführen.
🔄
Vercel Cache
Nach Deploy immer Hard-Refresh: Ctrl+Shift+R. Oder Vercel No-Cache Header (jetzt in vercel.json gesetzt).
🖥️
Beamer braucht gleichen Browser
Session-Info liegt in localStorage des Browsers. Beamer muss im selben Browser geöffnet werden wie das Dashboard. Kein Inkognito!
⏱️
Deploy-Zeit
Vercel braucht ~1–2 Min nach git push. Nicht sofort testen.
🔁
409 = OK
HTTP 409 von submit-exercise = idempotent (doppelter Submit). Als Erfolg behandeln, nicht als Fehler.
